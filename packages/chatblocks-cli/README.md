# chatblocks-cli

Install:

`npm install -g chatblocks-cli`
 
Run command
 
 `chatblocks --help`
  
  and follow instructions.
  
## What's inside?

- Express app and some predefined blocks
- Commands to run with nodemon and ngrok

## What's not?

- nodemon and ngrok
- Messenger App settings (page access token and verify token)
